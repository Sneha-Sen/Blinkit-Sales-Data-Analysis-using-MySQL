# Blinkit-Sales-Data-Analysis-using-MySQL

Analyzed Blinkit sales dataset to identify sales trends, product performance, and outlet-level insights using SQL queries. Performed data cleaning, aggregation, and calculated business metrics for actionable insights.
# Key Contributions:

-Cleaned inconsistent categorical data using CASE WHEN logic and standardization queries.

-Computed KPIs like total sales, average sales, average rating, and order count using SQL functions.

-Created outlet-level and category-level reports on sales distribution, fat content performance, and size-based contribution.

-Used conditional aggregation to replicate pivot-like summaries in MySQL.

-Derived percentage share of sales by outlet size and yearly performance trends for decision-making.

# Impact:
Simplified raw transactional data into interpretable insights that can drive business strategies for outlet performance and product mix optimization.
